Question-1: What are the different ways to select an element in the DOM?

There are 5 ways to select an elemen in the DOM.We can select by Id,class,tag etc.The methods are getElementById,getElementsByClassName ,getElementsByTagName ,querySelector ,querySelectorAll .

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML reading or modifying the HTML content inside an element.innertext reading or modifying visible text of an element .The textContent property retrieves or modifies all text within an element, including hidden elements.

Question-3: What is event delegation in the DOM?

Event delegation is a JavaScript technique where a parent element is used to manage events on its child elements.

Question-4: What is event bubbling in the DOM?

When an event occurs on an element, it first executes any event listeners attached to that element, then moves up to its ancestors, triggering their event listeners as well.
